### 预约
#### 短信验证码 vcode
* **请求地址**：/api/bespeak/vcode
* **请求方式**：get/post
* **请求参数**：
　　phone：手机号
* **响应结果**：
　　code：大于0表示成功
　　msg：错误描述
　　data：结果集
* **示例**：
　　http://127.0.0.1:8085/api/bespeak/vcode?phone=123456
　　{"code": 1, "msg": "成功"}

#### 参与扭蛋 save
* **请求地址**：/api/active/save
* **请求方式**：get/post
* **请求参数**：
　　phone：手机号
　　info：附加信息
* **响应结果**：
　　code：大于0表示成功
　　msg：错误描述
　　data：结果集
* **结果集Data说明**：
　　active：中奖记录pid
　　config：中奖的奖品
　　giftCode：中奖的奖品的礼包码，如果为空表示礼包码已用完。
* **示例**：
　　http://127.0.0.1:8085/api/active/save?phone=123456&info=
　　{"code": 1, "msg": "成功", "data": {active: 0, config: 0, giftCode:''}}
* **说明**：
　　响应结果data里的active是中奖记录的pid；config是所中奖项的pid，0表示为中奖。

### 新闻
#### 获取新闻列表 findByType
* **请求地址**：/api/news/findByType
* **请求方式**：get/post
* **请求参数**：
　　page：页面
　　rows：每页条数（最大100条）
* **响应结果**：
　　code：大于0表示成功
　　msg：错误描述
　　data：{"code": 1, "msg": "成功", data: [{...},...]}
* **示例**：
　　http://127.0.0.1:8085/api/news/findByType?page=1&rows=10
```javascript
{
	"msg": "成功 ",
	"total": 4,  //总条数
	"code": 1,
	"data": [{
		"date": "2019-01-14 00:38:46",  //日期
		"keywords": null,  //关键字（<meta> keywords） 
		"typeName": "新闻",  //分类名称
		"description": null,  //描述（<meta> content）
		"pid": 8,  //ID
		"recommend": 0,  //推荐  0--否，1--是
		"type": 1,  //分类
		"title": "test",  //标题
		"content": null,  //内容
		"site": "",  //链接地址
		"subtitle": "",  //副标题
		"illustration": "",  //标题插图
		"mark": 1,  //使用链接   0--否，1--是
		"status": 1  //状态  0：禁用，1：启用
	}, ...
	]
}
```
* **说明**：
返回倒序： 推荐 > 日期。
按界面布局，合理使用。
当mark=0，使用content内容进行显示；当mark=1，使用site跳转网址。
可用分类:
　　1　　新闻
　　2　　活动
　　3　　公告
　　4　　攻略
